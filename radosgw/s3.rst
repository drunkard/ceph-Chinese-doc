功能支持情况
------------
.. Features Support

下面的表格列出了对 Amazon S3 功能的支持情况：

+---------------------------------+---------+----------------------------------------+
| 功能                            | 状态    | 备注                                   |
+=================================+=========+========================================+
| **List Buckets**                | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Delete Bucket**               | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Create Bucket**               | 支持    | Different set of canned ACLs           |
+---------------------------------+---------+----------------------------------------+
| **Bucket Lifecycle**            | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Bucket Replication**          | 部分支持| 只允许跨域使用                         |
+---------------------------------+---------+----------------------------------------+
| **Policy (Buckets, Objects)**   | 支持    | ACL 和桶策略都支持                     |
+---------------------------------+---------+----------------------------------------+
| **Bucket Website**              | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Bucket ACLs (Get, Put)**      | 支持    | Different set of canned ACLs           |
+---------------------------------+---------+----------------------------------------+
| **Bucket Location**             | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Bucket Notification**         | 支持    | 见 `S3 通知兼容性`_                    |
+---------------------------------+---------+----------------------------------------+
| **Bucket Object Versions**      | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Get Bucket Info (HEAD)**      | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Bucket Request Payment**      | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Put Object**                  | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Delete Object**               | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Get Object**                  | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Object ACLs (Get, Put)**      | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Get Object Info (HEAD)**      | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **POST Object**                 | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Copy Object**                 | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Multipart Uploads**           | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Object Tagging**              | 支持    | See :ref:`tag_policy` for Policy verbs |
+---------------------------------+---------+----------------------------------------+
| **Bucket Tagging**              | 支持    |                                        |
+---------------------------------+---------+----------------------------------------+
| **Storage Class**               | 支持    | See :ref:`storage_classes`             |
+---------------------------------+---------+----------------------------------------+


不支持的 Header 字段
--------------------

下列的通用请求头部字段尚不支持：

+----------------------------+------------+
| 名称                       | 类型       |
+============================+============+
| **Server**                 | Response   |
+----------------------------+------------+
| **x-amz-delete-marker**    | Response   |
+----------------------------+------------+
| **x-amz-id-2**             | Response   |
+----------------------------+------------+
| **x-amz-version-id**       | Response   |
+----------------------------+------------+
